# 小程序图标指南

## 📱 小程序图标要求

### 图标规格：
- **尺寸**：144×144 像素（PNG格式）
- **格式**：PNG，支持透明背景
- **文件大小**：不超过 200KB

### 图标设计建议：

1. **简洁明了**：使用学位帽🎓或大脑🧠等与研究相关的元素
2. **配色方案**：使用蓝色(#4A6FA5)或紫色渐变，符合小程序主题
3. **圆角设计**：图标边缘最好有适当的圆角（微信会自动裁剪）
4. **文字清晰**：如果使用文字，确保在144px下清晰可读

### 图标示例设计：
```
背景：渐变蓝色 (#4A6FA5 → #3d5a80)
中心元素：学位帽🎓
底部文字：研值
```

## 🎨 如何添加图标

### 方法一：在微信开发者工具中添加
1. 打开微信开发者工具
2. 点击左侧「详情」→「基本信息」
3. 点击「小程序图标」右侧的「上传」按钮
4. 选择准备好的144×144 PNG图标
5. 点击「确定」保存

### 方法二：手动添加（不推荐）
微信小程序图标不需要手动放在项目目录中，直接在开发者工具中上传即可。
